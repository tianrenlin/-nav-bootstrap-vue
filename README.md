# Nav-bootstrap-vue
+ 本项目主要由Vue-cli3.0+Vux+bootstrap-vue搭建，使用本地静态数据进行模拟；
+ 请使用主流浏览器预览。虽是响应式页面，但有的浏览器并不怎么支持，所以有的显示效果不好。
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
